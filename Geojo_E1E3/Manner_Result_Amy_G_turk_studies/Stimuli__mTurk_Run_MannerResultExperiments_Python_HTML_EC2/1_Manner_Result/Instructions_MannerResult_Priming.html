Directions

In this study, you will be learning the meanings of made-up verbs. You will do so by watching videos which are described by sentences which contain these made-up verbs. You will also be asked to answer some questions.

For all of the verbs you learn, you will see:

  1. A sentence with a mystery verb, followed by the video it describes
  2. Two new videos. After each, you will decide if the event can be described using the mystery verb.

Only for verbs 1 to 8, you will see:

  1. Three new videos, each described by a sentence with the novel verb. Watch carefully! These will teach you the meaning of the mystery verb!!
  2. Two new videos. As before, you will decide if the mystery verb can describe the event
